This set of packages includes tools for efficiently working with the srcML format, including:

  • Convert source code to srcML, whether a single file, entire project, or code fragment (e.g., individual if statement)
  • Convert srcML back to source code without losing whitespace, formatting, etc.
  • Perform queries on source code using XPath and RelaxNG
  • Perform transformations on source code using XSLT

The first package includes the srcml command-line client, while the second package, called Development, enables the integration of srcML features into your own application.

For further details, visit srcML.org
